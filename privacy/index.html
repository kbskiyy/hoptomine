Данный сайт использует сервис Cloudflare для защиты от сетевых атак, фильтрации вредоносного трафика и ускорения загрузки страниц...

В процессе работы Cloudflare может автоматически обрабатывать техническую информацию, включая IP-адрес пользователя, а также устанавливать необходимые cookies для обеспечения безопасности соединения и корректной работы сайта...

Используемые cookies являются техническими и применяются исключительно для защиты сайта, предотвращения злоупотреблений и обеспечения стабильной работы сервиса...

Для корректной работы сайта используется localStorage для запоминания нажатия кнопки « ПОНЯТНО » на уведомлении о cookies, данные не передаются третьим лицам и применяются только для функциональности сайта...

Также сайт может использовать технические ресурсы и инфраструктуру сервисов Google, включая fonts.googleapis.com, который применяется для загрузки шрифтов...

В отдельных случаях для технической статистики производительности и безопасности может использоваться сервис cloudflareinsights.com ( static.cloudflareinsights.com ), являющийся частью инфраструктуры Cloudflare...

Администрация сайта HOPTOMINE не собирает, не хранит и не обрабатывает персональные данные пользователей напрямую...

Администрация сайта HOPTOMINE не передает персональные данные пользователей третьим лицам... При этом техническая информация ( например IP-адрес и данные браузера ) может автоматически обрабатываться сервисами инфраструктуры сайта, такими как Cloudflare и Google, которые используются для защиты, доставки контента и корректной работы сайта...
